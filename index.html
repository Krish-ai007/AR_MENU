Setup Guide
1

Connect Firebase

Create a free Firebase project → Firestore Database → open the HTML file, find the firebaseConfig section and paste your config. One time setup.

2

Add restaurants

Restaurants tab → + Add. Each restaurant is isolated — name, icon, colour, and its own dish list stored in Firestore.

3

Film dishes on black

Matte black cloth + turntable, rotate slowly 360° for 8-12 seconds. Upload to Cloudinary and paste the .mp4 URL.

4

Save dish — gets DB entry

Each dish is saved to Firestore. QR code links carry only the restaurant ID + dish ID — no video URL exposed. Customer scans → fetches from DB → sees dish in AR.

5

Customer experience

Scan QR → camera opens → tilt phone toward table → surface detected → dish appears on the table in AR. Drag to rotate 360°. Pinch to resize.

✦ Surface Detection

Customers see a "Tilt toward table" prompt. Using device gyroscope, when the phone is angled toward a flat surface the AR grid locks and the dish is placed on the detected plane — making it look like it's sitting on the actual table.

⚡ Firebase Rules (Firestore)

Set rules to allow read for all, write only for authenticated users. Restaurant data → restaurants/{id}. Dishes → restaurants/{id}/dishes/{dishId}.
